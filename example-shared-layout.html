How it works:
  1. Navbar is loaded automatically from layouts/admin-navbar.html
  2. Active page is highlighted based on data-page="dashboard"
  3. No need to copy navbar code - just use <div id="navbar-container"></div>
  4. Easy to maintain - update navbar once, applies everywhere!
Admin Layout

For admin dashboard pages

data-layout="admin"
Coach Layout

For coach dashboard pages

data-layout="coach"
Client Layout

For client dashboard pages

data-layout="client"
Code Example
Minimal HTML needed:
<!DOCTYPE html>
<html lang="en">
<head>
    <!-- CSS links -->
</head>
<body data-layout="admin" data-page="dashboard">
    
    <!-- Navbar Container -->
    <div id="navbar-container"></div>

    <!-- Your Content -->
    <div class="container-fluid px-4 py-4">
        <h2>My Page</h2>
    </div>

    <!-- Scripts -->
    <script src="js/layout-loader.js"></script>
</body>
</html>
